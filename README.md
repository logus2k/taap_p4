# TAAP Final Project

Project structure for Advanced Time Series Forecasting, Optimization and Explainability.

## Main components
- Baseline GRU forecasting model
- Evolutionary optimization
- Explainable AI (XAI)
- Efficiency and latency analysis
- TimeGAN-based synthetic data augmentation

## Structure
- `notebooks/` experiments and final notebook
- `src/data/` data loading and splitting
- `src/features/` feature engineering and windowing
- `src/models/` GRU models and training utilities
- `src/evolution/` evolutionary optimization
- `src/xai/` explainability methods
- `src/gan/` TimeGAN pipeline
- `src/utils/` metrics, profiling, helpers
- `results/` figures, tables and logs
